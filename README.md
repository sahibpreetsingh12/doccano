# Doccano Client setup
To use doccano client for doing operations on doccano using api.
To complete the work :-

# <font color='red'> Files </font>
- [x] Csv_2_db.py ---> Taking Data from CSV file and Storing it in DB
- [x] Db_2_doccano.py -----> Taking Data from DB to Doccano (Because uploading Data to doccano has limiation we can only upload 29 MB of documnets via UI at one Time)
- [x] Output_generator.py --

## First Install Doccano Client from [here](https://github.com/doccano/doccano-client)
Then in the same folder created save the files of this repo for their working

# For our Use case we have used Postgres DB 

To install Postgres DB in linux refer [here](https://www.digitalocean.com/community/tutorials/how-to-install-and-use-postgresql-on-ubuntu-18-04)

# Now to connect to Postgres Db using Python We used Psycopg2 package 
Install from [here](https://pypi.org/project/psycopg2/)
